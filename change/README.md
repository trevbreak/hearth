# Herth Changelog

This directory contains the changelog for all completed features and phases of Herth.

## Overview

When a phase or major feature is completed:
1. The plan file is moved from `/plan` to this changelog
2. Implementation details, dates, and decisions are documented
3. Bug fixes and breaking changes are recorded
